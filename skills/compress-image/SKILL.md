# Compress Image

Compression is the final export step, not the design step.

## Output

- WebP or optimized PNG/JPEG
- reduced asset size for publishing

## Rules

- keep source masters
- compress only after approval
- optimize derivatives, not the only original

## Workflow Integration

Apply after still generation and before final publishing handoff.
