# Cover Image

Cover images are not generic illustrations. They carry hierarchy, composition, and brand signal.

## Output

- 16:9 hero
- optional square and vertical variants

## Rules

- clear anchor object
- negative space for overlays
- one dominant metaphor
- consistent palette from the shared visual system

## Workflow Integration

Packs should emit `modules/cover-image-brief.md`.
The hero still used in Manim payoff scenes should usually come from this skill.
