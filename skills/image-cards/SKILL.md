# Image Cards

Break one idea into a short visual sequence.

## Output

- cover card
- 2-6 supporting cards
- closing CTA card

## Rules

- prompt-file-first
- card 1 establishes the visual anchor
- later cards inherit palette, character, and motif consistency
- each card has one clear informational purpose

## Workflow Integration

Packs should emit `modules/image-cards-brief.md`.
Use this skill when a blog post or lesson needs a shareable companion sequence.
